# JavaScript sample

## Running the sample

1. Install dependencies

    ```bash
    npm install
    ```

2. Rename `sample.env` to `.env` and fill in the values

    ```bash
    cp sample.env .env
    ```

3. Run the sample

    ```bash
    node search.js
    ```
